# PBA League
- 배방 리그

## Rules
- 게임비는 참여자 분담
- 게임당 각자 3개씩 알을 두고 시작 (게임 진행하면서 핸디캡 적용)
- 라운드별 승리자는 다음 라운드에 1개의 알 추가로 시작
- 5승 이상시 알 1개 추가
- 10승 이상시 알 2개 추가
- 빡은 없지만 예외적으로 공이 다이를 나가거나 직접 공을 건드는 경우 알 추가

## 장소
- 배방당구장

## 참가 선수
- 박종호 5알 `1위`
- 이재웅 4알
- 배윤하 4알
- 김혁인 4알
- 김원태 3알
