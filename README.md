# PBA_League
- 배방리그

## Rules
- 게임비는 참여자 분담
- 게임당 각자 3개씩 알을 두고 시작 (게임 진행하면서 핸디캡 적용)
- 라운드별 승리자는 다음 라운드에 1개의 알 추가로 시작

## 장소
- 배방당구장

## 참가 선수
- 이재웅 
- 배윤하
- 김원태
- 박종호 `1위`
- 김혁인

## 역대 기록
- 최다 우승 횟수 **1회** (이재웅, 박종호)
- 최다 연속우승 횟수 **1연속** (이재웅, 박종호)
- 최다 세트승점 우승 **8세트** (이재웅, 박종호)
- 제 1회 우승자 **이재웅** (8세트)
- 제 2회 우승자 **박종호** (8세트)
