# PBA League
- 배방 리그

## Rules
- 게임비는 참여자 분담
- 게임당 각자 3개씩 알을 두고 시작 (게임 진행하면서 핸디캡 적용)
- 라운드별 승리자는 다음 라운드에 1개의 알 추가로 시작

## 장소
- 배방당구장

## 참가 선수
- 박종호 5알 `1위`
- 이재웅 4알
- 배윤하 3알
- 김원태 3알
- 김혁인 3알

- 최다 세트승점 우승 **8세트** (이재웅, 박종호)
- 제 1회 우승자 **이재웅** (8세트)
